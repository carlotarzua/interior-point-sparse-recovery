# Provenance

## Project origin

This repository is a portfolio-focused refactoring of a MAT 387/487 university project on Interior Point Methods and Linear Programming.

The course project provided:

- the sparse recovery problem statement,
- the deterministic matrix and signal setup conventions,
- the Interior Point algorithm specification and parameter values,
- a MATLAB scaffold around the section where students implemented the main optimization steps.

## Work represented here

The portfolio repository presents:

- the implemented Interior Point iteration,
- modular refactoring of the MATLAB code,
- quantitative recovery metrics,
- result visualizations,
- technical documentation,
- explicit numerical-computing decisions.

## Why this statement is included

A professional portfolio should distinguish between:

1. the problem and scaffold supplied by a course, and
2. the algorithmic implementation, engineering work, analysis, and documentation completed by the student.

This repository intentionally makes that distinction visible.
